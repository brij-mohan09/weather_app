# 🌤️ JS Weather App

A modern, responsive weather application built with JavaScript and deployed on **Vercel**. It provides users with current weather conditions, 5-day forecasts, and detailed weather highlights like air quality and wind speed — all in one clean, interactive UI.

🎯 **Live Demo:** https://weather-app-sigma-cyan-30.vercel.app/

---

## 📌 Features

✔️ **Search by Location**  
Users can search for any city and instantly view the weather.

✔️ **Current Weather**  
Displays real-time data including temperature, humidity, visibility, and more.

✔️ **5-Day Forecast**  
Visual cards showing future weather trends for the next 5 days.

✔️ **Weather Highlights**  
Detailed metrics like air quality (AQI), pressure, wind speed, feels like temperature, and more.

✔️ **Hourly Forecast**  
Hour-by-hour breakdown of temperatures with time markers.

✔️ **Responsive Design**  
Looks great on mobile, tablet, and desktop screens.

---

## 🏗️ Built With

- **JavaScript (Vanilla)** – Core application logic  
- **HTML5 & CSS** – Markup and styling  
- **Weather APIs** – Real-time weather data  
- **Vercel** – Deployment and hosting
